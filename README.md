# jquery.loopmovement
一个循环移动的jquery插件

## 安装

安装之前先要引入jquery:

```html
<link rel="stylesheet" href="/path/to/jquery.loopmovement.css" />
<script src="/path/to/jquery.loopmovement.min.js"></script>
```

## 使用

使用非常简单:

```html
<script type="text/javascript">
	$('#box1').loopmovement({
		'direction':'top', 	//方向
		'speed':20			//速度,越小越快
	});
</script>
```
